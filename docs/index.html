Manifold Index Calculator

3D index, refined index, Dehn filling & non-closable cycles for hyperbolic 3-manifolds

v0.1.0
macOS · Apple Silicon (arm64) · macOS 11+
⬇ Download .app

First launch: right-click the app → Open → Open (Gatekeeper bypass).

3D Index

Compute I(m, e) q-series for any cusped manifold in the SnapPy census.

Refined Index

Two-variable (q, t) refinement with automatic basis selection.

Dehn Filling

Fill cusps with (P, Q) slopes and compute the filled index.

Non-closable Cycles

Search for slopes whose filled index vanishes identically.

Requirements

  • macOS 11 (Big Sur) or later
  • Apple Silicon Mac (M1 / M2 / M3 / M4)
  • No Python installation needed — everything is bundled

Getting Started

  • Download and unzip ManifoldIndex.zip
  • Move ManifoldIndex.app to your Applications folder
  • Right-click → Open → Open (first launch only)
  • Enter a manifold name (e.g. m004) and click Run

Build from Source

Requires Python ≥ 3.10 and the packages listed in pyproject.toml.

  • pip install -e ".[dev]"
  • manifold-index to launch the GUI
  • pytest tests/ to run the test suite
