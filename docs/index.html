Capabilities

Comprehensive tools for hyperbolic 3-manifold research

📐

3D Index

Compute I(m, e) q-series for any cusped manifold in the SnapPy census.

🔄

Refined Index

Two-variable (q, t) refinement with automatic basis selection.

🎯

Dehn Filling

Fill cusps with (P, Q) slopes and compute the filled index.

🔍

Non-closable Cycles

Search for slopes whose filled index vanishes identically.

📦

Data Packs

Pre-computed kernels and indices to skip expensive computations.

🚀

Cross-Platform

Native apps for macOS and Windows. No dependencies required.

Installation

macOS

Extract the zip and move ManifoldIndex.app to Applications. On first launch, macOS will block it. Clear the quarantine flag:

xattr -cr ~/Downloads/ManifoldIndex.app

Then launch normally. This is only required once.

Windows

Download the zip, extract it, and run ManifoldIndex.exe from the extracted folder. If Windows Defender warns, click More info → Run anyway.

From Source

Requires Python ≥ 3.10 and dependencies in pyproject.toml.

pip install -e ".[dev]"
manifold-index # Launch GUI
pytest tests/ # Run tests
