Installation

macOS

  1. Download ManifoldIndex.zip from the download section above.
  2. Open the zip — macOS expands it to ManifoldIndex.app. Drag the app into /Applications (or any folder you prefer).
  3. Because the app is not notarised, Gatekeeper will block the first launch. Clear the quarantine flag with:
    xattr -cr /Applications/ManifoldIndex.app
    Then double-click the app as normal. This step is only required once.
  4. If you skipped step 3 and already saw the "cannot be opened" dialog, open System Settings → Privacy & Security and click Open Anyway next to the ManifoldIndex entry.

The app bundles its own Python runtime and SnapPy — you do not need to install anything else.

Windows

  1. Download ManifoldIndex.exe from the download section above (it's a single self-contained executable — no installer, no zip).
  2. Move it anywhere you like (Desktop, Documents, a new folder — it's fine). Double-click to run.
  3. On first launch, Windows SmartScreen may show a blue pop-up saying "Windows protected your PC" because the binary isn't signed by a recognised publisher. Click More info, then Run anyway. You will only see this once per download.
  4. If Microsoft Defender or a third-party antivirus quarantines the file, add an exclusion for the folder you put ManifoldIndex.exe in. The installer-less format occasionally trips heuristic scanners.

Startup takes ~5–10 seconds on first launch while the embedded Python unpacks into %LOCALAPPDATA%\Temp. Subsequent launches are faster.

From Source

Requires Python 3.10 or newer. After cloning the repo:

pip install -e ".[dev]"
manifold-index # Launch GUI
pytest tests/ # Run test suite

Changelog

v1.0.4 — first stable release

  • Data Hub: 3-tab workflow (Download · Generate · Export & Share) with sequential multi-pack downloads, pause/resume/cancel generate queue, and a publish flow that infers category from file types.
  • Offline KaTeX rendering: math views no longer require network access.
  • Weyl compatibility diagnostic with collapsed-edge support, plus multi-cusp adjoint projection checks in the filling service.
  • v0.4-style α/β notation throughout the UI, with γᵢ/δᵢ display for NC cycles.
  • Cross-platform builds: macOS .app and Windows .exe from a unified release.sh, with GitHub Actions handling the Windows build.
  • Repository layout flattened — src/, tests/, assets/, and build scripts now live at the repo root.

v0.5.15

  • Display Improvements: Better fill result layout with combined metadata and improved LaTeX notation
  • Windows Support: Added build_app.bat script for standalone .exe building on Windows
  • New Widget: math_label.py for enhanced mathematical notation display
  • Code Quality: Simplified UI labels and reduced complexity in display components
  • Improved multi-cusp filling display with per-cusp slope formatting

v0.5.2

  • Critical Fix: Resolved SnapPy SQLite thread-binding error on manifold loading
  • Threading: Manifold loaded in main thread to avoid cross-thread database access
  • All manifolds from SnapPy census can now be loaded reliably

v0.5.1

  • UI Redesign: Complete v0.4-style formatters with mathematical notation (α, β, γ, δ)
  • KaTeX Rendering: Inline LaTeX throughout — table headers, parameters, results
  • Per-Cycle Weyl: Independent Weyl compatibility checks for each NC cycle basis choice
  • Enhanced Tables: NC cycle display with per-cycle Weyl vectors as tuples, professional appearance
  • Improved Display: Better fill result layout, transparent backgrounds, proper mathematical notation

v0.5.0

  • Performance: Yield CPU and release GIL in progress callbacks for macOS responsiveness
  • Stability: Comprehensive error analysis in data handling and worker management
  • Robustness: Enhanced session state handling with stale signal guards
  • 76+ tests passing

v0.4.2

  • Critical: Fixed half-integer e summation in Dehn filling
  • Multi-cusp Weyl symmetry corrections for accurate manifold analysis
  • Corrected Weyl convention: f(m,e) = f(−m,−e)
  • Stale data pack removal for data integrity

v0.4.0

  • Complete computation core rewrite with cleaner architecture
  • Full (½)ℤ² lattice support for IS chains
  • Pre-computed kernels-qq50 data pack fully recomputed
